Обновление ФИ: DLQ

Описание

Невалидные события обновления ФИ перенаправляются в DLQ.

Невалидными считаются события, в которых присутствует любое из следующего:

  • нарушен формат сообщения
  • есть поля без значений
  • есть неподдерживаемые поля
